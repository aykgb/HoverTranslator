# HoverTranslate (Ctrl + Hover 自动翻译)

**HoverTranslate** 是一款专为极致阅读体验设计的 Chrome 插件。无需点击，无需离开当前页面，只需悬停并按下键盘按键，即可获取高精度的中英互译结果。

## 🌟 核心特性

- **智能双模触发**：支持鼠标悬停在文字上按住 `Ctrl` 键自动识别翻译，或选中一段特定文本后按 `Ctrl` 触发。
- **中英智能互译**：内置语言检测逻辑，自动识别中英文并进行互译（英译中 / 中译英）。
- **极简交互设计**：
  - **加载反馈**：支持带呼吸动画的 `⏳ 翻译中...` 状态。
  - **智能消失**：翻译框在鼠标移出范围后自动消失，无需多余点击。
- **多款高级主题**：提供“高级勃艮第红”、“经典极客黑”、“简约护眼白”三款高对比度主题。
- **即时配置面板**：点击插件图标即可弹出下拉设置框，支持“用完即走”的自动关闭逻辑。
- **极高稳定性**：内置上下文失效探测与自销毁机制，彻底解决插件更新后的网页报错问题。

## 🛠️ 安装方法

1. 下载项目源代码。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目文件夹即可。

## 🎨 技术栈

- **Manifest V3**：遵循最新的 Chrome 插件标准。
- **Google Translate API**：使用稳定、快速的公共翻译接口。
- **Vanilla JS & CSS3**：零依赖，极致轻量，确保网页运行流畅。
